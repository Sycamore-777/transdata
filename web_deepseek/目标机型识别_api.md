## 2. 更新知识库
- 接口名：/api/update
- 请求方法：POST
- 参数名：file（文件为markdown，后缀为.md）
```
- 目前内置分割规则：
以# 为单个块的分割标识， #后面的文字作为相似性检索目标

- 文件内容示例：
```
